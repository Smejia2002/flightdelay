# Guion de Demo (60-120s)

1) Contexto rapido: queremos anticipar retrasos >= 15 minutos.
2) Mostrar el contrato del API (JSON simple).
3) Explicar el enriquecimiento de clima (API externa, T-2h).
4) Ejecutar /predict y mostrar la respuesta.
5) Cerrar con limitaciones y mejoras futuras.

## Texto sugerido (voz)
"Hoy mostramos FlightOnTime. Enviamos un JSON con aerolinea, origen, destino, fecha y distancia. Antes de predecir, el sistema agrega el clima con un servicio externo. El modelo responde si el vuelo sera puntual o retrasado, junto con la probabilidad. Esta es una primera version explicable; aun faltan mejoras como calibracion, monitoreo y retraining."
